# Word2Vec and Character-Level Name Generation

This repository contains implementations of:

- Word2Vec (CBOW and Skip-gram) trained on IIT Jodhpur corpus  
- Character-level name generation using  
  - Vanilla RNN  
  - Bidirectional LSTM (BLSTM)  
  - RNN with Attention  

## Requirements

Install the required libraries before running the notebooks:

pip install numpy pandas matplotlib nltk scikit-learn wordcloud torch torchvision beautifulsoup4 requests pdfplumber

## How to Run

1. Run `prob1.ipynb` for Word2Vec(CBOW and Skip-Gram) training and semantic analysis.
2. Run `prob2.ipynb` for character-level name generation models.
3. Execute all cells sequentially.
